\section{Design}

Argus is designed as a full-screen desktop application to utilise maximum screen space. The application is designed to be keyboard-driven, so keyboard shortcuts are displayed wherever possible. Additionally, the current instruction of the workflow is indicated on the top of the image in red to emphasise to annotators what input they are required to make. \cref{fig:dataset:argus:overview} shows the main \gls{ui}. Further segment-level features are captured in dialogs or user interactionÂ directly on the image, as shown in \cref{fig:dataset:argus:bib_and_face,fig:dataset:argus:prom_and_col}. We deployed Argus to data taggers remotely using the ClickOnce Deployment\footnoteurl{https://msdn.microsoft.com/en-us/library/t71a733d.aspx}{11 August 2017} strategy.
