\section{Design}

Argus is designed as a full-screen application to utilise maximum screen space. The application is designed to be keyboard-driven, so shortcuts are displayed wherever possible. Additionally, the current instruction of the workflow is indicated on the top of the image in red to stand out to image taggers. \cref{fig:dataset:argus:overview} shows the main \gls{ui}. Further segment-level features are captured in dialogs or user interactionÂ directly on the image, as shown in \cref{fig:dataset:argus:bib_and_face,fig:dataset:argus:prom_and_col}.
